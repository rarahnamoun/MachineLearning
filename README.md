# Machine Learning Course Projects

## Overview
This repository contains a comprehensive collection of machine learning projects covering fundamental algorithms, implementation from scratch, and practical applications. The coursework progresses from basic supervised learning algorithms to advanced deep learning and unsupervised learning techniques.

## Repository Structure

### 📁 [Homework 1: Decision Trees and K-Nearest Neighbors](./hw1/)
**Focus**: Fundamental supervised learning algorithms
- **Files**: `ml_hw1_q1_*.py`, `ml_hw1_q2_*.py`
- **Topics**: Decision Tree implementation, KNN classifier, data preprocessing
- **Datasets**: Loan approval prediction, Adult income classification
- **Key Skills**: Algorithm implementation from scratch, evaluation metrics, hyperparameter tuning

### 📁 [Homework 2: Linear Regression and Support Vector Machines](./hw2/)
**Focus**: Linear models and optimization techniques
- **Files**: `ML_HM2_Q1.ipynb`, `ML_HM2_Q2.ipynb`
- **Topics**: Linear regression, SVM implementation, kernel methods
- **Key Skills**: Gradient descent, quadratic programming, regularization techniques

### 📁 [Homework 3: Neural Networks and Multilayer Perceptrons](./hw3/)
**Focus**: Deep learning fundamentals and neural network implementation
- **Files**: `Q1.ipynb`
- **Topics**: Feedforward networks, backpropagation, customer segmentation
- **Key Skills**: Neural network architecture, gradient computation, optimization algorithms

### 📁 [Homework 4: Convolutional Neural Networks](./hw4/)
**Focus**: Computer vision and deep learning for image classification
- **Files**: `Q1.ipynb`
- **Topics**: CNN architecture, image processing, Oxford 102 Flower Dataset
- **Key Skills**: Convolutional operations, feature extraction, transfer learning

### 📁 [Homework 5: Clustering and Unsupervised Learning](./hw5/)
**Focus**: Unsupervised learning algorithms and data exploration
- **Files**: `Q1.ipynb`
- **Topics**: K-means, DBSCAN, hierarchical clustering, customer segmentation
- **Key Skills**: Cluster analysis, dimensionality reduction, evaluation metrics

## Getting Started

Each homework folder contains detailed README files with specific instructions, implementation details, and usage examples. Start with HW1 for foundational concepts and progress through the sequence for a comprehensive machine learning education.

### Prerequisites
- Python programming experience
- Basic linear algebra and calculus knowledge
- Statistics and probability fundamentals
- Curiosity for machine learning and data science

### Installation
```bash
# Clone the repository
git clone <repository-url>

# Install required dependencies
pip install numpy pandas matplotlib seaborn scikit-learn jupyter

# Navigate to specific homework folder
cd hw1/  # or hw2/, hw3/, hw4/, hw5/

# Launch Jupyter notebook
jupyter notebook
```
